# File Reader App

## Description

File Reader App is a simple JavaFX desktop application that allows users to select a text file and display its contents inside the application.

This project introduces:

* JavaFX GUI development
* FileChooser
* File handling
* BufferedReader
* Event handling

---

## Features

* Open a file from the computer
* Read text file content
* Display content in a TextArea
* Simple and user-friendly interface

---

## Technologies Used

* Java 17+
* JavaFX
* FileChooser
* BufferedReader

## How It Works

1. Launch the application
2. Click "Open File"
3. Select a .txt file
4. The file content is displayed inside the application
