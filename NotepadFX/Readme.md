# NotepadFX

A simple text editor built with JavaFX.

## Features

* Create a new document
* Open text files
* Save text files
* Edit text
* User-friendly graphical interface

---

## Technologies

* Java
* JavaFX
* FileChooser
* BufferedReader
* BufferedWriter

---

## Project Structure

```text
NotepadFX/

├── src/

└── README.md
```

---

## How to Run

### Clone project

```bash
git clone https://github.com/yourusername/notepadfx.git
```

### Compile

```bash
javac Main.java
```

### Run

```bash
java Main
```
