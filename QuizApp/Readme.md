# QuizApp JavaFX

## Description

QuizApp est une application graphique développée avec JavaFX.

Elle permet à l'utilisateur de répondre à une question à choix multiples et affiche immédiatement le résultat.

Ce projet a pour objectif de découvrir :

* JavaFX
* Les interfaces graphiques
* Les événements
* Les RadioButtons
* Les ToggleGroups

---

## Technologies

* Java 17+
* JavaFX

## Fonctionnalités

### Affichage d'une question

L'utilisateur voit une question avec plusieurs réponses possibles.

### Sélection d'une réponse

Une seule réponse peut être sélectionnée grâce à un ToggleGroup.

### Validation

Après avoir cliqué sur "Valider", l'application vérifie la réponse.

### Affichage du résultat

L'utilisateur est informé immédiatement si sa réponse est correcte ou incorrecte.
